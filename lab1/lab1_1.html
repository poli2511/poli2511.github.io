Для бесплатного хостинга веб-страниц мы будем использовать сервис GitHub Pages. Для этого необходимо создать аккаунт Google. С помощью этого аккаунта зарегистрируйтесь на GitHub. Создайте репозиторий username.github.io, где username - ваш логин на GitHub. Создайте в репозитории файл index.html и добавьте в него заголовок первого уровня "Лабораторные работы". Для удобной работы с репозиторием можно использовать GitHub.dev. Пришлите на e-mail admin@fedyaevys.ru ссылку на ваш сайт (вида username.github.io) с указанием полного ФИО и группы.
